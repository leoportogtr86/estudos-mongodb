Como conectar ao MongoDB usando o driver Node.js?
Use const { MongoClient } = require('mongodb'); const client = new MongoClient('mongodb://localhost:27017');.
Como criar um banco de dados no MongoDB usando Node.js?
Use const db = client.db('meuBancoDeDados');.
Como criar uma coleção no MongoDB usando Node.js?
Use db.createCollection('minhaColecao');.
Como inserir um único documento no MongoDB usando Node.js?
Use db.collection('minhaColecao').insertOne({nome: "João", idade: 25});.
Como inserir múltiplos documentos no MongoDB usando Node.js?
Use db.collection('minhaColecao').insertMany([{nome: "Ana", idade: 22}, {nome: "Pedro", idade: 28}]);.
Como consultar todos os documentos de uma coleção no MongoDB usando Node.js?
Use db.collection('minhaColecao').find().toArray();.
Como consultar um documento específico no MongoDB usando Node.js?
Use db.collection('minhaColecao').findOne({nome: "Ana"});.
Como atualizar um único documento no MongoDB usando Node.js?
Use db.collection('minhaColecao').updateOne({nome: "Ana"}, {$set: {idade: 23}});.
Como atualizar múltiplos documentos no MongoDB usando Node.js?
Use db.collection('minhaColecao').updateMany({idade: {$lt: 30}}, {$set: {status: "jovem"}});.
Como excluir um único documento no MongoDB usando Node.js?
Use db.collection('minhaColecao').deleteOne({nome: "Pedro"});.
Como excluir múltiplos documentos no MongoDB usando Node.js?
Use db.collection('minhaColecao').deleteMany({idade: {$lt: 25}});.
Como conectar ao MongoDB usando Java?
Use MongoClient mongoClient = new MongoClient(new MongoClientURI("mongodb://localhost:27017"));.
Como criar um banco de dados no MongoDB usando Java?
Use MongoDatabase database = mongoClient.getDatabase("meuBancoDeDados");.
Como criar uma coleção no MongoDB usando Java?
Use database.createCollection("minhaColecao");.
Como inserir um único documento no MongoDB usando Java?
Use collection.insertOne(new Document("nome", "João").append("idade", 25));.
Como inserir múltiplos documentos no MongoDB usando Java?
Use collection.insertMany(Arrays.asList(new Document("nome", "Ana").append("idade", 22), new Document("nome", "Pedro").append("idade", 28)));.
Como consultar todos os documentos de uma coleção no MongoDB usando Java?
Use collection.find().into(new ArrayList());.
Como consultar um documento específico no MongoDB usando Java?
Use collection.find(eq("nome", "Ana")).first();.
Como atualizar um único documento no MongoDB usando Java?
Use collection.updateOne(eq("nome", "Ana"), new Document("$set", new Document("idade", 23)));.
Como atualizar múltiplos documentos no MongoDB usando Java?
Use collection.updateMany(lt("idade", 30), new Document("$set", new Document("status", "jovem")));.
Como excluir um único documento no MongoDB usando Java?
Use collection.deleteOne(eq("nome", "Pedro"));.
Como excluir múltiplos documentos no MongoDB usando Java?
Use collection.deleteMany(lt("idade", 25));.
Como conectar ao MongoDB usando C#?
Use var client = new MongoClient("mongodb://localhost:27017");.
Como criar um banco de dados no MongoDB usando C#?
Use var database = client.GetDatabase("meuBancoDeDados");.
Como criar uma coleção no MongoDB usando C#?
Use database.CreateCollection("minhaColecao");.
Como inserir um único documento no MongoDB usando C#?
Use collection.InsertOne(new BsonDocument { { "nome", "João" }, { "idade", 25 } });.
Como inserir múltiplos documentos no MongoDB usando C#?
Use collection.InsertMany(new List { new BsonDocument { { "nome", "Ana" }, { "idade", 22 } }, new BsonDocument { { "nome", "Pedro" }, { "idade", 28 } } });.
Como consultar todos os documentos de uma coleção no MongoDB usando C#?
Use var documents = collection.Find(new BsonDocument()).ToList();.
Como consultar um documento específico no MongoDB usando C#?
Use var document = collection.Find(new BsonDocument("nome", "Ana")).FirstOrDefault();.
Como atualizar um único documento no MongoDB usando C#?
Use collection.UpdateOne(new BsonDocument("nome", "Ana"), Builders.Update.Set("idade", 23));.
Como atualizar múltiplos documentos no MongoDB usando C#?
Use collection.UpdateMany(new BsonDocument("idade", new BsonDocument("$lt", 30)), Builders.Update.Set("status", "jovem"));.
Como excluir um único documento no MongoDB usando C#?
Use collection.DeleteOne(new BsonDocument("nome", "Pedro"));.
Como excluir múltiplos documentos no MongoDB usando C#?
Use collection.DeleteMany(new BsonDocument("idade", new BsonDocument("$lt", 25)) );.
