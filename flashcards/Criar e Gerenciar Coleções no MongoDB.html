Operações Básicas no MongoDB - Criar e Gerenciar Coleções no MongoDB

Como criar uma coleção implicitamente no MongoDB?
Insira um documento em uma coleção que ainda não existe: db.minhaColecao.insertOne({ nome: "Alice", idade: 30 }).
Como criar uma coleção explicitamente no MongoDB?
Use o método createCollection: db.createCollection("minhaColecao").
Como criar uma coleção limitada no MongoDB?
Use o método createCollection com opções: db.createCollection("minhaColecaoComOpcoes", { capped: true, size: 5242880, max: 5000 }).
Como listar todas as coleções em um banco de dados no MongoDB usando a linha de comando?
Use o comando show collections.
Como listar todas as coleções em um banco de dados no MongoDB usando JavaScript no shell?
Use o método db.getCollectionNames().
Como renomear uma coleção no MongoDB?
Use o método renameCollection: db.minhaColecao.renameCollection("novaNomeDaColecao").
Como excluir uma coleção no MongoDB?
Use o método drop: db.minhaColecao.drop().
Como verificar se uma coleção foi removida no MongoDB?
Liste todas as coleções novamente usando show collections.
Como criar um índice em uma coleção no MongoDB?
Use o método createIndex: db.minhaColecao.createIndex({ nome: 1 }).
Como listar todos os índices em uma coleção no MongoDB?
Use o método getIndexes: db.minhaColecao.getIndexes().
Como excluir um índice em uma coleção no MongoDB?
Use o método dropIndex com o nome do índice: db.minhaColecao.dropIndex("nome_1").
Como definir regras de validação ao criar uma coleção no MongoDB?
Use o método createCollection com um validador: db.createCollection("minhaColecaoValidada", { validator: { $jsonSchema: { bsonType: "object", required: ["nome", "idade"], properties: { nome: { bsonType: "string", description: "deve ser uma string e é obrigatório" }, idade: { bsonType: "int", minimum: 0, description: "deve ser um inteiro não negativo e é obrigatório" } } } } }).
Como atualizar regras de validação em uma coleção existente no MongoDB?
Use o comando collMod: db.runCommand({ collMod: "minhaColecao", validator: { $jsonSchema: { bsonType: "object", required: ["nome", "idade"], properties: { nome: { bsonType: "string", description: "deve ser uma string e é obrigatório" }, idade: { bsonType: "int", minimum: 0, description: "deve ser um inteiro não negativo e é obrigatório" } } } } }).
O que é uma coleção limitada no MongoDB?
Uma coleção limitada tem um tamanho fixo e atua como um buffer circular.
Como definir o tamanho máximo de uma coleção limitada no MongoDB?
Use a opção size ao criar a coleção: db.createCollection("minhaColecaoComOpcoes", { capped: true, size: 5242880, max: 5000 }).
Como definir o número máximo de documentos em uma coleção limitada no MongoDB?
Use a opção max ao criar a coleção: db.createCollection("minhaColecaoComOpcoes", { capped: true, size: 5242880, max: 5000 }).
Como criar uma coleção com validação JSON Schema no MongoDB?
Use o método createCollection com um validador JSON Schema: db.createCollection("minhaColecaoValidada", { validator: { $jsonSchema: { bsonType: "object", required: ["nome", "idade"], properties: { nome: { bsonType: "string", description: "deve ser uma string e é obrigatório" }, idade: { bsonType: "int", minimum: 0, description: "deve ser um inteiro não negativo e é obrigatório" } } } } }).
Como adicionar um índice ascendente em um campo no MongoDB?
Use o método createIndex com 1 como valor: db.minhaColecao.createIndex({ nome: 1 }).
Como adicionar um índice descendente em um campo no MongoDB?
Use o método createIndex com -1 como valor: db.minhaColecao.createIndex({ nome: -1 }).
Como remover todos os documentos de uma coleção no MongoDB sem remover a coleção?
Use o método deleteMany sem filtro: db.minhaColecao.deleteMany({}).
Como garantir que um campo é único em uma coleção no MongoDB?
Crie um índice único no campo: db.minhaColecao.createIndex({ nome: 1 }, { unique: true }).
Como criar um índice composto no MongoDB?
Use o método createIndex com múltiplos campos: db.minhaColecao.createIndex({ nome: 1, idade: -1 }).
